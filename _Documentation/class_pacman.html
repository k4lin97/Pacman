◆ move()

void Pacman::move ( Map *  map)

Function that moves pacman.

Function checks in which direction pacman wants to move. There are three cases:

  • wall - pacman cannot move there,
  • scorePoint - pacman moves there and gets the point from map,
  • fruit - pacman moves there, gets the point from map and his ability to eat a ghost is set to true, If pacman moves number of score points on map is being deducted.
    Parameters
    mapmap on which pacman moves.
