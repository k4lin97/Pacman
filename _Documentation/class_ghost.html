◆ move()

void Ghost::move ( )

This function moves a ghost.

The function imitates the movement of ghost from a real pacman. For a ghost, the distance to its target (from its next position) is calculated. The ghost cannot go back, and if the distance to the target is equal, priority is given to: up, left, down, and right.
